EPL Web Service Endpoints

The following APIs are provided for supporting command line processes on the ILS:

  • POST /getuserpin/{user_id} or curl -d "user_id=21221012345678" http://server:6003/getuserpin
  • GET /status returns the status of the web service.
  • GET /version shows the version of the API.
Use Chrome or Firefox for best results, but some apis don't have web pages. Use cURL, Postman, or similar.
